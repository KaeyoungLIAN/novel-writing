# 📖 Novel Writing

> 每一个故事，都值得被认真书写。

## 项目简介

这是一个个人小说创作项目。在这里，我记录灵感、构建世界、塑造角色、编织情节。

## 📂 目录结构

```
novel-writing/
├── outline/        # 故事大纲与章节规划
├── chapters/       # 正文章节
├── characters/     # 角色设定与关系图谱
├── world/          # 世界观设定
└── notes/          # 灵感笔记与草稿
```

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/KaeyoungLIAN/novel-writing.git

# 开始创作
cd novel-writing
```

## ✍️ 创作规范

- 每章一个独立 Markdown 文件
- 角色设定统一放在 `characters/` 目录
- 灵感碎片随时记录在 `notes/` 中
