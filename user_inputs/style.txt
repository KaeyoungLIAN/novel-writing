# 写作风格
# 请在此文件中描述你希望小说采用的写作风格。
# 包括但不限于：语言风格、叙事视角、节奏特点、对话风格、描写偏好等。

# ===== 示例 =====

【语言风格】
简洁、干净的中文。句子不宜过长，多用短句与破折号制造停顿感。
少用成语和华丽修辞，偏白描，用具体的物象和动作来传递情绪。

【叙事视角】
第三人称有限视角。主视角跟随林墨，但每章可切换一次视角到苏晚。
不采用全知叙事，读者只看到主角能看到、能感知到的东西。

【节奏特点】
整体偏慢，不急不躁。重要场景可以充分展开，日常场景一笔带过。
对话不追求戏剧性，而是贴近真实生活中的沉默、欲言又止和词不达意。

【对话风格】
简洁、克制。人物说话符合身份：林墨话少但精准，苏晚语速快但偶尔走神，
陈师傅说话带本地口音，喜欢用反问句。对话不必完整，允许断句和沉默。

【描写偏好】
多用具象的感官描写——特别侧重触觉和嗅觉（木头腐朽的气味、梅雨季节的潮湿、
青苔的触感等）。少用大段的风景描写，将环境融入人物的动作和感知中。

【整体定位】
偏文学性的通俗小说。故事有可读性，但文字追求质感。
不追求情节上的反转和意外，而是让读者沉浸在古城渐渐消逝的氛围中。
