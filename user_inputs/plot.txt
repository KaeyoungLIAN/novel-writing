# ===== 示例 =====

【故事背景】
一座有1300年历史的江南古城，因旅游业过度开发和年轻人口外流而日渐衰败。
市政府计划启动新城改造项目，古城区面临大规模的拆旧建新。
在城市发展与文化保护之间，一场无声的拉锯战正在上演。

【核心冲突】
林墨所在的古城修复站接到了最后一批抢救性修复任务——要在新城改造启动前，
完成对古城区十二处文物点的测绘和修复记录。而苏晚的摄影项目《消逝的城》，
则试图用影像留存这些即将消失的街巷与生活。

两位主角都以自己的方式在与时间赛跑，
但他们都渐渐意识到，有些东西不是修好了、拍下来了就能留下的。

【关键转折点】
1. 林墨在修复一座老宅时发现了墙壁夹层中藏有一批民国时期的书信，
   揭示了这座古城在抗战时期的一段秘密历史
2. 苏晚拍摄的一张照片意外在网上走红，引发了外界对古城命运的广泛关注
3. 修复站因资金短缺被迫停工，林墨面临是否要接受外地高薪工作的抉择
4. 陈师傅在一次修缮中受伤，林墨独自承担起剩下的修复工作
5. 市政府召开新城改造听证会，居民意见严重分裂

【主题思想】
关于"何为真正的保护"：是保留建筑的外壳，还是延续生活在其中的人的记忆？
关于"告别"：每个人都终将与自己珍视的事物告别，但告别的姿态各有不同。

【情感基调】
温暖中带一丝苍凉。不煽情，不刻意悲壮，而是平静地讲述一个关于时间与选择的故事。
结尾是开放的：古城有了新的可能，但主角们也各自做出了自己的选择。

【字数预期】
10章，每章约8000字，总计约8万字的中篇小说。
